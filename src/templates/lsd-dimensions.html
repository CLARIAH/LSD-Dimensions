API calls

Returns the dimension URIs whose labels contain the specified query string.

GET http://grlc.io/api/albertmeronyo/lsd-queries/dimensions?query={query-string}

Returns the code URIs of codelists linked to the specified dimension URI.

GET http://grlc.io/api/albertmeronyo/lsd-queries/codes?dim={url-encoded-uri}
